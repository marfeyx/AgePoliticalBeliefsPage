# AgePoliticalBeliefsPage

Animated multi-page website about how age shapes voting behavior in the United States.

## Pages

- `index.html` — landing page with hero animation, counters, and high-level story.
- `insights.html` — generation-level behavior analysis and priority signals.
- `timeline.html` — historical trendline and scenario outlook.
- `about.html` — methodology and source references.

## Assets

- `style.css` — shared visual system, responsive layout, and animation styles.
- `script.js` — mobile nav, reveal-on-scroll, counters, cursor glow, particle effects, and tilt interactions.
